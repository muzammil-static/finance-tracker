# Personal Finance Tracker

## Overview
A responsive web application to manage personal finances, track income/expenses, view reports using charts, and ensure data persistence using XAMPP.

## Technologies
- HTML, CSS, JavaScript
- Chart.js
- PHP / Node.js (for backend)
- XAMPP
